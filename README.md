# Magergram Api (Server)

For Instagram clone on Node.js, GraphQL, Prisma (TypeScript)

[Go to Magergram client repository](https://github.com/magersoft/magergram-client)

#### Start server in dev mode
`yarn dev`

#### Build and compressing
`yarn build`

#### Runtime
`yarn start`
